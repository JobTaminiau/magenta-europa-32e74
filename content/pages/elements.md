### Inline Code and Code Blocks

Blocks of code are either fenced by lines with `three back-ticks`, or are indented `with four spaces`.

    <!-- Some example CSS code -->
    h1 {
        font-size: 2.5rem;
    }

Inline `code` has `back-ticks around` it.
